# TikHub Cloudflare Pages 部署说明

## 🚀 快速部署

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)
2. 点击左侧菜单的 "Pages"
3. 点击 "Create a project"
4. 选择 "Upload assets"
5. 上传此文件夹中的所有文件
6. 点击 "Deploy site"

## 🌐 域名配置

部署完成后：
1. 点击 "Custom domains"
2. 添加你的域名
3. 按照提示配置DNS记录

## ⚙️ 环境变量

如果需要后端API支持，请设置：
- `API_BASE_URL`: 你的后端API地址

## 📞 技术支持

如有问题，请查看项目文档或联系技术支持。
